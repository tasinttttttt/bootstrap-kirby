## Kirby 3 bootstrap

Commands and configuration files to speed things up.

## Project

Requirements: `git, node`

Project structure

```
.
├── .editorconfig
├── .gitignore
├── Makefile
├── README.md
└── src
```

## Installation
```sh
npm install
```

## Commands

### Development starts a php server `http://localhost:8000` (edit port in `.rollup.config.js`)
```sh
npm run dev
```

### Build
```sh
npm run build
```
